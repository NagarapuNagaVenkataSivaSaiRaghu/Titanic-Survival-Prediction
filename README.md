# Titanic Survival Prediction

## Project Overview

This project predicts whether a passenger survived the Titanic disaster using machine learning classification techniques.

## Objectives

* Perform data cleaning and preprocessing
* Handle missing values and categorical features
* Conduct exploratory data analysis (EDA)
* Train and evaluate classification models
* Predict passenger survival outcomes

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Project Workflow

1. Data Cleaning and Preprocessing
2. Exploratory Data Analysis
3. Feature Engineering
4. Model Training
5. Model Evaluation
6. Prediction Generation

## Results

The model was trained and evaluated using classification metrics to predict passenger survival.

* Data Cleaning
* Feature Engineering
* Classification Modeling
* Data Visualization
* Machine Learning Fundamentals
